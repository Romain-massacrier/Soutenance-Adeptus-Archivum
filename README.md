Adeptus Archivum

Application web immersive inspirée de l’univers Warhammer 40,000, permettant de consulter des factions, des archives et des décrets impériaux.

Objectif du projet :
Ce projet a été réalisé dans le cadre d’une soutenance.

L’objectif est de :

- Concevoir une interface moderne à partir d’une maquette Figma
- Intégrer cette interface en HTML/CSS (mobile-first)
- Structurer un projet web propre et évolutif
- Appliquer les bonnes pratiques de développement (Git, organisation, séparation des responsabilités)

Structure du projet :

adeptus-archivum/
│
├── index.html        # écran d’introduction
├── accueil.html      # page principale
├── factions.html     # sélection des factions
│
├── css/

│
├── images/           # assets visuels
│
├── js/
│   └── script.js     # interactions (prévu pour évolution)
│
├── README.md
└── .gitignore
